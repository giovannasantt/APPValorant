# APPValorant
## Feito por: Giovanna Santana

Aplicativo baseado na Valorant-API (https://dash.valorant-api.com/?fix), feito para verificar imagens e informações dos agentes do jogo de maneira prática e intuitiva. Projeto feito para avaliação de aplicativos mobile.
